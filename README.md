## 📖 Sobre o projeto

Este repositório foi desenvolvido como resposta a um desafio proposto pelo professor **Ewerton**, na disciplina de **Gestão Corporativa de TI** da FIAP.

O objetivo do projeto é aplicar, na prática, os conceitos abordados em sala de aula, contribuindo para o desenvolvimento de habilidades técnicas e analíticas relacionadas à gestão de tecnologia da informação em ambientes corporativos.

Além disso, este trabalho possui caráter avaliativo, podendo ser utilizado para composição de nota em atividades futuras.

---

## 🎯 Objetivos

* Aplicar conceitos de Gestão Corporativa de TI
* Desenvolver raciocínio crítico na resolução de problemas
* Praticar organização e versionamento de código com Git e GitHub
* Consolidar conhecimentos adquiridos em aula
